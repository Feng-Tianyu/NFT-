# 去中心化NFT拍卖平台

#### 运行方式：

1.安装truffle

```
npm install -g truffle
```

2.打开ganache，创建私链

![image-20211106041951102](D:\Edgedownload\dacheng\去中心化NFT拍卖平台\image-20211106041951102.png)

3.编译合约

```
truffle compile
```

4.部署合约

```
truffle migrate
```

5.运行

```
npm start
```

注：

​	如果遇到无法铸造的情况，可以尝试科学上网
